# Sample Output Directory

This directory will contain generated PDFs and sidecar YAML files when you run the invoice generator.

## Structure

-  - Generated invoice PDFs (ignored by git)
-  - Machine-readable sidecar files with calculated data
-  - GSTR-1 JSON exports for GST filing

## Note

Generated files in this directory are excluded from version control via  except for sample YAML files which may be included for reference.
